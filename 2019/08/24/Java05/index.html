Variables and scopes

Local variables

  • Local variables are variables defined within a function.
  • Local variables mush be initialized before use, no Java default value。
  • Life time of local variables are within this method.

Instance variables/ member variables

  • Instance variables can be ether initialized by constructor or by Java default value.
  • Life time of instance variables are within the instance.
  • Visibility is defined by their access modifiers.

Class variables

  • Class variables belongs to a class not a instance.
