Java仅有8种数据类型, 其余均为class

  • byte: 可以用来表示年龄 Max: 0111,1111 (127)/ Min: 1000,0000 (-128) (对应正数的二进制取反+1, 比最大正数多一个).
  • short: 16-bit (最左一位表示正负)
  • int: 32-bit/ 4-byte (很多情况下不够用, 可能会溢出)
  • long: 64-bit/ 8-byte (一般情况下溢出不了)
  • float: 32-bit float (10^38, 最高分辨率10^-46) float是科学计数法, 含有指数位, 所以比int能表示的大得多
  • double: 64-bit float (更常用)
  • char: 16-bit (能表示2^16个字符, 涵盖主流常见字符集)
  • boolean: 1-byte true/ false
    1-byte 是计算机读写的最小基本单元

Java的取整

  • 结果的精度以参与运算中精度较高的那个为准

Char

  • 和 Python不同，需要区分单引号和双引号
  • Char 只能表示一个字符，必须使用单引号
  • String 用来表示任意长度的字符，必须使用双引号

进制

  • 默认表示十进制
  • 在左侧加上‘0b’表示二进制
  • 在左侧加上‘0’ 表示八进制
  • 在左侧加上‘0x’表示十六进制

字符串拼接

  • “abc” + 1 + 7 –> “abc17”
  • 1 + 7 + “abc” –> “8abc”

Operators

  • Postfix: i++ / i– (变量本身加1, 表达式是增加之前的值)

  • Unary: ++i / –i (变量本身加1, 表达式是增加之后的值) / +i(正)/ -i(负)/ ~i(按位取反)/ !i(逻辑取反)

  • Multiplicative: * / %

  • additive : + / -

  • shift << / >> (算术右移) / >>> (逻辑右移 高位补0)

  • relational: < / > / <= / >= / instanceof(左边Object 右边类, 表示左边Object是不是类的instance)

  • equality: == / !=

  • bitwise AND: x & y (x的二进制int 和y的二进制int 按位取与)

  • bitwise exclusive OR: x ^ y (x的二进制int 和y的二进制int 按位取异或 不同为真相同为假)

  • bitwise inclusive OR: x | y (x的二进制int 和y的二进制int 按位取或)

  • logical AND: x && y (x和y都是boolean 逻辑取与)

  • logical OR: x || y (x和y都是boolean 逻辑取或)

  • ternary (三步运算符): x = 1 > 0 ? 7 : 6 (冒号两侧必须为相同数据类型)

  • assignment: =/ +=/ -=/ …
