Class and Object

Object

  • Object can be anything, it contains:
  1. states 状态 (attribute, properties, characteristics…)
  2. behavior 行为

Class

  • Java class describes a the state and behavior requirements of a set of objects of the same kind.
  • class 是个描述，描述清楚想要的 requirement是什么.
  • Java 中所有通过 new创建出来的 Objects都存储在 heap中.

Array

  • Creating an array is similar to creating an object.
  • 数组是线性结构，每个元素在内存中紧密相连，长度不能变化
  • 想要动态变化数组大小，使用ArrayList
  • 数组中存储的所有元素，必须都是同一类型！
