Class & Obj

Primitive types and Class types:

  • Primitive 只有八种，其他所有都是 Class types

存在Stack 还是Heap？

  • 局部变量 - 存在 stack上
  • 成员变量(instance variable) - 存在 Heap上

Constructor:

  • 函数名与类名相同，大写字母开头
  • 没有return type

this

  • 指当前object
  • 如果instance variable 和 local variable 名字重名 就必须写this.

Null

  • An empty reference, point to no ref.

Static

  • 可以修饰 field 或 method， 说明这个field/ method是属于整个类的 而不单单是具体这个对象的。
  • Static函数 只能访问 Static变量；
  • 非 Static函数 既可以访问 Static变量，也可以访问 非Static变量；

Access modifier

  • public – 全局可见
  • protected – 只有子类和同个package可见
  • default – 只有同个package可见
  • private – 只在同一个类里可见

Stacks 和 Heaps

  • Stack 存局部变量
  • Heap 存new出来的成员变量
  • 从Stack 到Heap 就是一个寻址的过程
